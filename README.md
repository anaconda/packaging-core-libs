### Part 1: Build and run with library Version 1.0

   ```bash
   make lib-v1
   make app
   make run
   ```

### Part 2: Build and run with library Version 2.0

   ```bash
   make lib-v2
   make app
   make run
   ```

### Part 3: Run with library Version 3.0 without recompiling

   ```bash
   make lib-v3
   make run
   ```
